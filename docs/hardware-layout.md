# ハードウェア レイアウト

ロボット座標系におけるホイール（ステッピングモーター）と下向きカメラの位置・向きを記述する。

## ロボット座標系

- **+x**: ロボット前方
- **+y**: ロボット左方向
- **+z**: 上方向
- **角度**: +x から反時計回り（CCW）を正とする

## ホイール配置

3つのオムニホイールを120°等間隔に配置する。
取り付け位置角 φ は +x（前方）から反時計回りに計測する。

```
              前方 (+x)
                ↑
            [W0] φ=0°
           ／       ＼
          ／           ＼
  左 (+y)←               →右 (−y)
          ＼           ／
           ＼         ／
   [W2] φ=240°   [W1] φ=120°
```

| モーター | 取り付け位置角 φ | モーター軸方向（半径方向） | ローラー転がり方向（接線方向） |
|---|---|---|---|
| motor[0] | 0°（前方） | 0° ／ 180°（前後） | 90° ／ 270°（左右） |
| motor[1] | 120°（左後方） | 120° ／ 300° | 210° ／ 30° |
| motor[2] | 240°（右後方） | 240° ／ 60° | 330° ／ 150° |

- **中心半径**: 50 mm（ロボット中心からホイール接地点まで）
- **ホイール半径**: 24 mm（直径 48 mm）

## 下向きカメラ配置

カメラはロボット中心に下向きに固定する。

- **FOV**: 102°
- **画像上端**: ロボット前方（+x）に対応
- **画像右端**: ロボット右方向（−y）に対応

```
        画像上端 = ロボット前方 (+x)
               ↑
        ┌──────────────┐
        │              │
        │              │
左(+y) ←│    カメラ     │→ 右(−y)
        │    (俯瞰)    │
        │              │
        └──────────────┘
               ↓
        画像下端 = ロボット後方 (−x)
```

### VisionModule との対応

`DownwardVisionModule` はカメラ画像を前後左右の4ストリップに分割して壁を検出する。
上記のカメラ搭載向きにより、各ストリップとロボット座標系の対応は以下の通り。

| ストリップ | 画像上の位置 | ロボット方向 | 世界座標方向（East 向き時） |
|---|---|---|---|
| front | 画像上端 | +x（前方） | East |
| back | 画像下端 | −x（後方） | West |
| left | 画像左端 | +y（左方） | North |
| right | 画像右端 | −y（右方） | South |

> ロボットは常に East（θ=0）を向いて探索走行するため、ロボット前方 = 世界東方向 となる。
